## RAG (Retrieval Augmented Generation)

Conectar o modelo a arquivos de conhecimento para que responda com base nesses dados. Muito mais eficiente que Fine Tuning.

## Fine Tuning

Retreinar o modelo para mudar o comportamento. Usar apenas para:

- Mudar estilo permanentemente
- Ensinar um padrão específico de resposta
- Especializar o modelo (Jurídico, Médico, etc)
