#ia #bestpractices #sop 
É uma forma de avaliar a qualidade das respostas geradas por uma IA, considerando os seguintes aspectos:

- Intent: se está respondendo o que foi perguntado
- Complete: se cobre todas as solicitações do prompt
- On-Truth: se todas os fatos são corretos
- No Contradictions: se não há contradições
- Relevance: a resposta é relevante para o que foi perguntado?
- Efficient: se a resposta fica se repetindo em algum conceito
- Articulate: gramática e formatação
- Length: se a resposta é extensa demais
